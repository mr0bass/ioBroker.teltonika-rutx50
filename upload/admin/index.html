Information

This adapter enables SMS sending through Teltonika RUTX50 router via its local REST API.

Features:
  • Send SMS messages through router's cellular connection
  • Blockly integration for visual scripting
  • Connection status monitoring
  • Signal strength monitoring
  • Message history tracking
Blockly Blocks:
  • Send SMS block for visual scripting
  • SMS status check blocks
